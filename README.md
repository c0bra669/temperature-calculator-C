# temperature-calculator-C
Simple temperature conversion calculator writen in C.
# instalation linux

```bash
git clone https://github.com/c0bra669/temperature-calculator-C.git
sudo gcc -o /bin/calc temperature-calculator-C/main.c
```


# Usage 
```bash
calc $number$unit
```
$number - intager, can be negative
$unit:
- K - Kelvin
- C - Celsius
- F - Farenheit
### use CAPITAL LETTERS for unit!
